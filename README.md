# everland_purchase

Purchase package
: Process that purchase tickets (ticket price is different by ticket type, age group and discount type)
: 1. user inputs information
: 2. program find raw price and calculate result price
: 3. save order list to file

Statistic package
: Statistic work with report file that saved order list
: 1. read all data of file
: 2. do statistic process and get result - save them to map or list
: 3. print results in console and save to file
